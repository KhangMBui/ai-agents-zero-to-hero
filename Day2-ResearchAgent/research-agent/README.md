**⭐ Project Choice 1 — “Research & Summarizer Agent” (Fastest Intro)**

A simple agent that:

1. Takes a topic

2. Plans steps on how to research it

3. Calls a search tool

4. Reads web results

5. Summarizes them

6. Saves the summary to LTM (JSON file)

**Tools used:**

- search() tool

- file write tool

- json memory tool

**Teaches:**

- ReAct

- Dynamic planning

- STM/LTM loop

- Tool calling reliability

- Memory writing

**Difficulty:** ⭐⭐ (easy)
**Time to finish:** ~45 minutes
**Best for:** Day 3 warm-up
